LBW - Little Bit Wiser in Cricket

Welcome to LBW, the term generally refers to leg before wicket in the sport of cricket (see here in Wikipedia), but I decided to give it a fun turn, using LBW to be short of Little Bit Wiser. You'll see projects related to cricket, the sport I follow, over here.

Projects

Feel free to check these projects out here, and you can also access them within each other using the menu bar

  • Batting Statistics - This contains some of the batting basics, such as a batter's strike rate, and their average.
  • Bowling Statistics - This contains some of the bowling basics, such as a bowler's economy rate, strike rate, and their average.
  • NRR - This assists in calculating the net run rate of a team in a cricket match.

This was created using JavaScript, HTML, and CSS. The repository, containing the code is below in the footer.
